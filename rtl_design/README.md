# RTL Design

This folder contains the Verilog HDL implementation of the BFloat16 Floating-Point Processor.

Modules included:
- Unpacker
- Comparator
- Aligner
- Math (addition/subtraction)
- Multiplier
- Normalizer
- Packer
- Main
- implement

Together, these modules implement BFloat16 addition, subtraction, and multiplication operations.
